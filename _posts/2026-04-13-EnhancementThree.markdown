<h2>The Enhancement </h2>

![MongoDB](/images/Enhancement3MongoDB.png)

The artifact is one I can use to complete the Database component of the ePortfolio by using drivers and CRUD to connect the software to the MongoDB database. Some of the skills of this showcases is the creation and understanding of API’s, being able to get the frontend and backend to work together and illustrate my understanding of noSQL databases. To access the database, the user must input the username and password when prompted. This increases security: an authenticated username and password will bring the user to the main menu, while an invalid one will end the program. The original artifact is greatly improved through this enhancement, as it now can save changes made while the program is running. This way, even after the software has shut down, it will retain updates upon restarting.
