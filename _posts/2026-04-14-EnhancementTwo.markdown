Implementing a Binary Search Tree into the C++ application

The artifact is the result of enhancement one for the computer science capstone, the updated search and rescue animal system rewritten in C++. It can intake new dogs and monkeys and store them in a vector. The animals can then be displayed to the user. It also allows animals to be reserved by type and service location.

<h2>The Enhancement </h2>

![BST Driver](/images/Enhancement2Code1.png)

This artifact was one I could use to complete the Algorithms and Data Structures component of the ePortfolio by implementing a binary search tree to store and search for animals more effectively than simply in a vector. This showed my understanding of time complexity and Big O notation, creating data structures using C++, and creative solutions for working with more than one data set.
