Rewriting The Grazioso Search and Rescue Appliction from Java to C++.

The original artifact was Java program written to serve as an animal search and rescue system. The system can intake new dogs or monkeys and display ones currently in the system. It is also able to reserve an animal based on animal type and location upon user request. This artifact was created as a final project in my IT 145: Foundations in Application Development course. I took this class at the beginning of my time at SNHU.
